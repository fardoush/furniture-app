# 🛋️ Home Decor Furniture App

A modern and responsive furniture & home decor website built with React and Tailwind CSS.  
Users can explore different types of furniture, view product details, and enjoy a clean UI/UX experience.

🔗 Live Website: https://home-decor-furniture-app.netlify.app/


---

## 🚀 Features

- 🏠 Beautiful Home Page with modern UI
- 🪑 Furniture product listing
- 📂 Category-based browsing (Living Room, Bedroom, etc.)
- 🔍 Product details view
- 📱 Fully responsive (Mobile, Tablet, Desktop)
- ⚡ Fast loading and smooth user experience

---

## 🛠️ Technologies Used

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3
